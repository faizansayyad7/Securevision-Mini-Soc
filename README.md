# 🛡️ SecureVision - Mini SOC Dashboard

A modern **Security Operations Center (SOC) Dashboard** built with **Python, Flask, SQLite, HTML, CSS, JavaScript, and Chart.js**.

This project analyzes security log files, detects suspicious activities, calculates risk scores, and provides an interactive dashboard for monitoring security events.

---

## 🚀 Features

- 🔐 User Authentication (Login & Logout)
- 📂 Upload Security Log Files
- 📊 Interactive Dashboard
- 📈 Threat Analytics Charts
- 🚨 Incident Detection
- ⚠️ Risk Score Calculation
- 🔍 Log Search & Severity Filter
- 📄 Export Logs to CSV
- 🗃️ SQLite Database Integration
- 📋 Incident Summary
- 📊 Security Health Indicator

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/faizansayyad7/Securevision-Mini-Soc.git
```

Open project

```bash
cd Securevision-Mini-Soc
```

Create Virtual Environment

```bash
python -m venv .venv
```

Activate Virtual Environment

Windows

```bash
.venv\Scripts\activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Application

```bash
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```
